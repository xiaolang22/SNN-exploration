# 实验11：自适应结构进化拓扑实验
#### 实验目的
在当前六模式刺激分类主实验的随机连接网络基础上，引入自适应结构进化拓扑筛选机制，考察进化得到的连接拓扑是否能够在训练前就提供更好的刺激响应分离性，并进一步分析其与 STDP 联合作用时对最终分类性能的影响。

#### 原理简介
本实验参考 Pan 等人在 2023 年关于 recurrent SNN/LSM 自适应结构进化的思路，在正式训练前先对网络拓扑进行短程评估与筛选。具体做法是将连接拓扑视为可进化个体，以邻接结构为染色体，通过静默神经元修复、局部重连、热点桥接重连等方式产生后代，并用短程代理适应度函数对候选拓扑进行选择。

本实验在现有框架中实现的是“结构进化拓扑筛选”部分；论文中提到的 DA-BCM 规则未在本实验中引入。实验中的可塑性部分仍采用原框架中的 STDP。

此外，为了区分“进化拓扑的优势究竟来自拓扑组织本身，还是仅仅来自度分布变化”，本实验额外设置了 degree-matched random 对照组。该组保留进化拓扑的入度和出度统计，但将具体连边重新随机化，用于检验拓扑组织结构本身的贡献。

#### 实验内容
以主实验为基础，完成如下六组实验：

1. `01_random_no_plasticity`
   Random topology + no plasticity
2. `02_evolved_no_plasticity`
   Evolved topology + no plasticity
3. `03_random_stdp`
   Random topology + STDP
4. `04_evolved_stdp`
   Evolved topology + STDP
5. `05_degree_matched_random_stdp`
   Degree-matched random topology + STDP
6. `06_degree_matched_random_no_plasticity`
   Degree-matched random topology + no plasticity

所有组均保持与主实验尽可能一致的网络规模、刺激参数和分类流程；实验11只增加拓扑进化、degree-matched 随机化和相关适应度参数。最终结果采用五轮调参循环中表现最均衡的第三轮参数与输出结果，即 `cycle03`。

#### 实验假设
1. 经过结构进化筛选的网络，在训练前就具有更好的刺激响应分离性。
2. `Evolved topology + STDP` 的最终分类准确率高于 `Random topology + STDP`。
3. 结构进化可以提高神经元利用率，降低静默神经元比例，并减少极端同步爆发。
4. 进化拓扑的优势来自拓扑组织本身，而不是来自度分布变化。

#### 实验结果与分析
##### 一、拓扑进化筛选结果（短程代理评估）

结构进化阶段共运行 12 代，在第 7 代首次找到最优个体，最终选择的拓扑标签为 `gen06_offspring`。下面给出随机基线、进化拓扑和 degree-matched 随机拓扑在短程代理评估中的关键指标：

| 拓扑类型 | fitness | separation score | decode score | plasticity decode score | silent neuron ratio | burst penalty |
| --- | --- | --- | --- | --- | --- | --- |
| baseline random | `0.7613` | `0.7010` | `0.7083` | `0.9167` | `30.10%` | `0.0192` |
| evolved topology | `0.9065` | `0.8460` | `0.7917` | `1.0000` | `26.40%` | `0.0263` |
| degree-matched random | `0.6765` | `0.6181` | `0.6667` | `0.7500` | `31.60%` | `0.0200` |

**分析**：
1. 与随机基线相比，进化拓扑在 `fitness`、`separation score`、`decode score` 和 `plasticity decode score` 上均明显提高，说明拓扑进化确实在正式训练前筛选出了更有利于刺激响应分离的结构。
2. 进化拓扑的静默神经元比例由 `30.10%` 降至 `26.40%`，说明神经元利用率有所提升。
3. 但 `burst penalty` 从 `0.0192` 上升到 `0.0263`，说明当前适应度函数虽然提升了分离性和可解码性，却没有同步地压低爆发倾向。
4. degree-matched random 的各项指标均明显落后于进化拓扑，表明进化拓扑的优势不是简单来自度分布匹配，而是来自更优的连边组织方式。

这部分结果支持假设 1，并为假设 4 提供了直接证据。

##### 二、第 3 轮分类结果（最终分类准确率）

以下结果取三轮正式实验中的第 3 轮，作为最终分类结果：

| 组别 | raw count | V100 |
| --- | --- | --- |
| `01_random_no_plasticity` | `79.17%` | `64.17%` |
| `02_evolved_no_plasticity` | `89.17%` | `79.17%` |
| `03_random_stdp` | `84.17%` | `74.17%` |
| `04_evolved_stdp` | `84.17%` | `84.17%` |
| `05_degree_matched_random_stdp` | `81.67%` | `66.67%` |
| `06_degree_matched_random_no_plasticity` | `80.00%` | `71.67%` |

**分析**：
1. 对比 `02_evolved_no_plasticity` 与 `01_random_no_plasticity`，进化拓扑在无可塑性条件下实现了明显提升，`raw count` 提高 `10.00` 个百分点，`V100` 提高 `15.00` 个百分点，说明仅凭拓扑组织优化就能增强分类性能。
2. 对比 `04_evolved_stdp` 与 `03_random_stdp`，两组在 `raw count` 上持平，均为 `84.17%`；但 `04_evolved_stdp` 在 `V100` 上从 `74.17%` 提升到 `84.17%`。这说明进化拓扑与 STDP 的联合作用主要改善了强刺激条件下的稳定分类表现。
3. 对比 `04_evolved_stdp` 与 `05_degree_matched_random_stdp`，进化拓扑组在 `raw count` 和 `V100` 上分别高出 `2.50` 和 `17.50` 个百分点；对比 `02_evolved_no_plasticity` 与 `06_degree_matched_random_no_plasticity`，分别高出 `9.17` 和 `7.50` 个百分点。说明即使度分布被匹配，随机化后的网络也无法复现进化拓扑的性能优势。
4. 对比 `03_random_stdp` 与 `01_random_no_plasticity`，STDP 在随机拓扑上提高了最终表现；但对比 `04_evolved_stdp` 与 `02_evolved_no_plasticity`，加入 STDP 后 `raw count` 反而下降，而 `V100` 上升。这说明当前实现下的 STDP 更偏向于重塑网络动态和高刺激响应稳定性，而不一定单调提升总体六分类准确率。

这部分结果表明：假设 2 得到部分支持。若以 `V100` 为代表性指标，则 `Evolved topology + STDP` 明显优于 `Random topology + STDP`；若以最终 `raw count` 为唯一标准，则两者并未拉开差距。

##### 三、功能性指标与同步性变化

第 3 轮实验的功能连接统计如下：

| 组别 | avg correlation | global efficiency | modularity |
| --- | --- | --- | --- |
| `01_random_no_plasticity` | `0.0843` | `0.5000` | `0.0784` |
| `02_evolved_no_plasticity` | `0.0970` | `0.5541` | `0.0850` |
| `03_random_stdp` | `0.2444` | `0.7522` | `0.0300` |
| `04_evolved_stdp` | `0.2627` | `0.7896` | `0.0457` |
| `05_degree_matched_random_stdp` | `0.2426` | `0.7709` | `0.0347` |
| `06_degree_matched_random_no_plasticity` | `0.1104` | `0.6164` | `0.0768` |

**分析**：
1. STDP 组整体具有更高的 `avg correlation` 和 `global efficiency`，说明在线可塑性会增强网络中的协同活动和全局传播能力。
2. 同时，STDP 组的 `modularity` 普遍低于无可塑性组，说明在反复刺激和权重更新后，网络响应更容易朝整体相关化方向发展。
3. 这一现象能够解释为什么 `Evolved topology + STDP` 虽然在 `V100` 上表现最好，但在总体 `raw count` 上并未继续超过 `Evolved topology + no plasticity`。原因是当前实现中的 STDP 为无监督、全程在线更新，它能强化高刺激下的稳定传播，却也可能压缩不同刺激模式之间原本已经较好的全局可分性。

因此，假设 3 只得到部分支持：结构进化确实降低了静默神经元比例，但并未同时降低爆发惩罚，且在与 STDP 联合后网络相关性明显上升。

##### 四、对实验假设的综合判断

1. **假设 1：成立。**
   进化拓扑在短程代理评估中的分离性和可解码性指标均优于随机基线。
2. **假设 2：部分成立。**
   `Evolved topology + STDP` 在 `V100` 上优于 `Random topology + STDP`，但最终 `raw count` 未超过对方。
3. **假设 3：部分成立。**
   结构进化降低了静默神经元比例，但没有同步降低爆发惩罚，且 STDP 会进一步提高相关性。
4. **假设 4：成立。**
   无论在有无 STDP 的条件下，进化拓扑均明显优于 degree-matched random，对照结果表明优势来自拓扑组织本身，而非单纯度分布变化。

##### 五、不完全满足假设的原因分析

1. 当前结构进化优化的是短程代理适应度，而正式实验是更长时间尺度下的完整刺激流程，二者目标并不完全一致。
2. 现有可塑性机制仅包含经典 STDP，未引入论文中的 DA-BCM 或奖励调制项，因此权重变化主要由时序相关性驱动，而不是直接由分类目标驱动。
3. 在已经具备较好结构分离性的进化拓扑上，无监督 STDP 可能会进一步强化共激活通路，造成模式间响应趋同，从而削弱总体 `raw count` 分类优势。
4. 当前适应度函数虽然已经加入 `plasticity decode score` 和活动度约束，但对“抑制同步化/爆发”的约束仍然偏弱，因此会出现静默神经元减少、但相关性和爆发倾向同时上升的折中结果。

#### 结论
实验11表明，在当前六模式 BNN/SNN 框架下，引入自适应结构进化拓扑机制是有效的。进化得到的拓扑在正式训练前就表现出更高的刺激响应分离性，并在最终实验中稳定优于普通随机拓扑；同时，degree-matched 对照组证明，这种优势主要来自拓扑组织本身，而不是简单的度分布变化。

从最终分类结果看，进化拓扑在“无可塑性”条件下已经能显著提升性能；在与 STDP 联合时，则主要体现为高刺激条件下分类稳定性的增强。另一方面，当前实现中的 STDP 尚未与分类目标完全对齐，因此 `Evolved topology + STDP` 并未在所有指标上都超过 `Evolved topology + no plasticity`。

综合来看，实验11已经较好支持了“结构进化拓扑有效”以及“其优势来自拓扑组织本身”这两条核心结论；但“结构进化与 STDP 联合后必然全面优于其他组”这一点目前只得到部分支持，后续若要进一步提升效果，应优先优化可塑性目标与同步性约束，而不是再去改动主实验的基础参数。
